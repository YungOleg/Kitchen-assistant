# KA - Kitchen Assistant

KA is your professional culinary assistant. With it, you will always find the perfect recipe for any occasion, whether it's a dinner for two or a festive dinner for the whole family.
---
### Tech-stack
* 100% Kotlin 
    * Coroutines
    * Flow
* Jetpack Compose
* Android Jetpack
    * Room
    * Hilt*
* Retrofit2
* Coil
### Architecture
* Layers architecture
* MVVM
